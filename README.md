# Make NexusMods Readable Again

## How to install

[![Install directly with Stylus](https://img.shields.io/badge/Install%20directly%20with-Stylus-00adad.svg)](https://raw.githubusercontent.com/fabd/nexusmods-stylus/master/nexusmods-readable.user.css)

You'll want to install the **Stylus addon** first*: [Firefox](https://addons.mozilla.org/en-US/firefox/addon/styl-us/), 
 [Google Chrome](https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne?hl=en).

Tested on Desktop, Firefox, best for 1000px wide and up.
